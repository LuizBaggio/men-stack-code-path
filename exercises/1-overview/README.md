#Inicializar MongoDB

##Windows
```
$ mongo.exe
```

##MacOS
```
$ mongo
```

##Linux
```
$ mongo
```

#Comandos básicos

##Visualizar todas as databases
```
$ show dbs
```

##Criar uma database
```
$ use databaseName
```

##Navegar entre databases
```
$ use databaseName
```

##Visualizar database atual
```
$ db
```

##Navegar entre databases
```
$ use databaseName
```

##Criar collection
```
$ db.createColletion(“collectionName”)
```
